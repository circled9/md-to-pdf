# Markdown to PDF

MarkdownをPDFに変換するやつです。

## Getting Started

docs以下に **.md** を置いて `npm run print` すると、いい感じに **.pdf** にしてくれます。

### Prerequisities

node.js and npm is required.

### Installing

1. install dependencies.
```bash
npm install
```

2. put your .md file to docs.
```bash
touch docs/some_markdown.md
vi docs/some_markdown.md
```

3. run print task
```bash
npm run print
```
